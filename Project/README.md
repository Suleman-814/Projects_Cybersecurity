##  Project Overview

 Directory Enumeration – Discover hidden directories on web servers
 File Downloader – Download files using Python
 Hash Cracker – Crack hashes using wordlists
 Keylogger – Logs keystrokes (educational use only)
 Network Scanner – Identify active devices on a network
 Port Scanner – Scan for open ports
 SSH Brute Forcing – Test SSH login credentials
 Subdomain Enumeration – Discover subdomains
